# stack-attack-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that Stack Attack helper and lander

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
